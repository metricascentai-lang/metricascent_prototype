# MetricAscent Website

Marketing landing page for MetricAscent — helping independent window &amp; door companies
capture more opportunities, book more estimates, and grow revenue.

## Structure

- `index.html` — the complete single-page site (self-contained: inline CSS + JS)
- `assets/` — logo and hero images (must stay alongside `index.html`)

External dependencies are loaded from CDNs at runtime: Google Fonts (Sora / IBM Plex Sans / Mono)
and [Lucide](https://lucide.dev) icons. An internet connection is required for fonts and icons.

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
